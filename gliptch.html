Portfolio of Graham P Heath

GliPtcH

Inspired by the simple process of opening an image in a text editor, making a change, and in so doing, breaking the original image but producing a sometimes beautiful byproduct. Ths is available as a live web app, Java executeable or an OSX Application.

The hardest part of this was creating a work flow by which the broken image would get rendered. Accessing its byte array, or in the Java version, bit array, I had the data but I still needed to stitch it back together. In code sets this was accomplished by creating a fake stream of data, which could then be assembled as an image.
